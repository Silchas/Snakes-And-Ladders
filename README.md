# Snakes-And-Ladders
## Build a Python Script to play Snakes and Ladders versus four players.
### Details
<p>
  The program is a Snakes And Ladders game.</br>
  The game utilizes the randint module from the random library to simulate dice rolling.</br>
  The program creates two instances of snakes and ladders and connects numbers that when the player lands on, they either move up or down on landing on a ladder or a snake.</br>
</p>
